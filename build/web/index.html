This project is about developing a Web Application that can solve the problems faced by the students. With the implementation of this project work, the students will be able to select the college of their interest based on faculty and location. Users can easily access the application computer connected to the Internet using a standard browser.

We are developing a web application about college recommendation using Java as a programming language. Our purpose to develop this application is to provide facilities for student to select the college as per their requirements. And this application helps to recommend the college if the requirements are matched.

College recommendation is a web application where the students can fill-up the form. Once they are registered then they can search and get information about different college.
